# Compensation Benefits Scrum Project

## Description
This project aims to create a system for managing compensation and benefits within an organization, focusing on efficient and secure management. The project includes a frontend interface and a robust backend, which incorporates security features to ensure safe access and management.

### Features Implemented So Far:
- **Frontend**: The user interface has been developed with necessary components for viewing and interacting with compensation and benefits information.
- **Security**: Integrated security features to ensure authorized access using JWT authentication. Roles-based access control has been implemented to distinguish between user and admin privileges.
